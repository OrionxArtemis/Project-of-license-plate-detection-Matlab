This is our final-term project of object detection.
It could be run in Matlab and used to detect license plate in any given image.

Made by Võ Đăng Tài, Lê Văn Thái, Nguyễn Việt Long.
Completed in December 2021.


INTRODUCTION

Extract 'Database.zip' first.

Run 'main.m' in Matlab to run all programs.

Click 'Train' button in GUI to train all images in path 1, save all trained data and calculate Accuracy.

Click 'Test' button in GUI and select an image in your device to check if it contains 1-line license plate, 2-line license plate or not and draw its bounding box around its license plate.
